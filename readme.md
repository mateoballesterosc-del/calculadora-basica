# Calculadora en Python

Este es mi primer repositorio publicado en GitHub.

## Descripción

La aplicación permite realizar operaciones básicas:

- Suma
- Resta
- Multiplicación
- División

## Herramientas usadas

- Python 3
- Git
- GitHub

## Cómo ejecutar el programa

1. Descargar el repositorio.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo calculadora.py con Python.

	python calculadora.py
